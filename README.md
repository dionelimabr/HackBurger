# 🍔 HackBurger

Ecommerce de hambúrgueres inspirado na arquitetura do OWASP Juice Shop, desenvolvido propositalmente com vulnerabilidades para treinamento em AppSec.

## 🚀 Como Rodar Localmente

1. Clone o repositório.
2. Copie `.env.example` para `.env` e ajuste se necessário.
3. Rode `npm run start` para subir os containers via Docker Compose.

- **Frontend**: http://localhost:4200
- **Backend API**: http://localhost:3000/api
- **Grafana**: http://localhost:3001
- **Prometheus**: http://localhost:9090

> **Aviso**: Este projeto contém vulnerabilidades de segurança intencionais. NÃO DEPLOIE EM PRODUÇÃO.
